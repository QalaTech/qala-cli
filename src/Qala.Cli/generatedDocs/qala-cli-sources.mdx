## `list`
this command lists all the sources available in your 
environment.

**Examples:**
```sh
qala sources ls
```


## `name`
this command retrieves the source with the specified 
NAME.

**Options:**
<table>
<tr><th>Option</th><th>Description</th></tr>
</table>
**Examples:**
```sh
qala sources name <NAME>
```


## `create`
this command creates a new source for the Qala 
CLI.

**Options:**
<table>
<tr><th>Option</th><th>Description</th></tr>
<tr><td>`algorithm`</td><td>The algorithm of the source when authentication type is 
jwt (RSA or HSA).</td></tr>
<tr><td>`apiKeyName`</td><td>The name of the api key when authentication type is api 
key.</td></tr>
<tr><td>`apiKeyValue`</td><td>The value of the api key when authentication type is api 
key.</td></tr>
<tr><td>`audience`</td><td>The audience of the source when authentication type is 
jwt.</td></tr>
<tr><td>`a` / `authenticationType`</td><td>The authentication type of the source (Basic, ApiKey or 
JWT).</td></tr>
<tr><td>`d` / `description`</td><td>The description of the source.</td></tr>
<tr><td>`i` / `ip-whitelisting`</td><td>The comma separated list of the ips allowed to access the
source.</td></tr>
<tr><td>`issuer`</td><td>The issuer of the source when authentication type is 
jwt.</td></tr>
<tr><td>`m` / `methods`</td><td>The comma separated list of the http methods available to
the source.</td></tr>
<tr><td>`n` / `name`</td><td>The Name of the source.</td></tr>
<tr><td>`password`</td><td>The password of the source when authentication type is 
basic.</td></tr>
<tr><td>`publicKey`</td><td>The public key of the source when authentication type is 
jwt and algorithm is RSA.</td></tr>
<tr><td>`secret`</td><td>The secret of the source when authentication type is jwt 
and algorithm is HSA.</td></tr>
<tr><td>`username`</td><td>The username of the source when authentication type is 
basic.</td></tr>
</table>
**Examples:**
```sh
qala sources create -n <NAME> -d  <DESCRIPTION> -m <METHODS_COMMA_SEPERATED> -i  <IP_WHITELISTING_COMMA_SEPERATED> -a <AUTHENTICATION_TYPE>
```
