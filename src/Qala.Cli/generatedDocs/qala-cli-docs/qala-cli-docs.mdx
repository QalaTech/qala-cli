### `login`
this commands initiates the process for the user to 
login.

**Examples:**
```sh
login
```


### `config`
this command creates enables the configuration of the Qala CLI 
(for automation purposes).

**Options:**
<table>
<tr><th>Option</th><th>Description</th></tr>
<tr><td>`e` / `env`</td><td>The environment ID to use for the request.</td></tr>
<tr><td>`k` / `key`</td><td>The API key to use for the request.</td></tr>
</table>
**Examples:**
```sh
config -k <API_KEY> -e <ENV_ID>
```
