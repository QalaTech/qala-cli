### `list`
this command lists all the subscriber groups available in 
your environment.

**Examples:**
```sh
qala subscriber-groups ls
```


### `inspect`
this command retrieves the subscriber group with the 
specified NAME.

**Options:**
<table>
<tr><th>Option</th><th>Description</th></tr>
</table>
**Examples:**
```sh
qala subscriber-groups inspect  <SUBSCRIBER_GROUP_NAME>
```


### `create`
this command creates a new subscriber group for the Qala 
CLI.

**Options:**
<table>
<tr><th>Option</th><th>Description</th></tr>
<tr><td>`a` / `audience`</td><td>The audience to scope the subscriber group.</td></tr>
<tr><td>`d` / `description`</td><td>The description of the subscriber group.</td></tr>
<tr><td>`n` / `name`</td><td>The name of the subscriber group.</td></tr>
<tr><td>`t` / `topics`</td><td>The comma separated list of topic names.</td></tr>
</table>
**Examples:**
```sh
qala subscriber-groups create -n <NAME> -d  <DESCRIPTION> -t <TOPICS_COMMA_SEPERATED_NAMES> -a <AUDIENCE>
```


### `update`
this command updates an existing subscriber group for the 
Qala CLI.

**Options:**
<table>
<tr><th>Option</th><th>Description</th></tr>
<tr><td>`a` / `audience`</td><td>The audience to scope the subscriber group.</td></tr>
<tr><td>`d` / `description`</td><td>The description of the subscriber group.</td></tr>
<tr><td>`n` / `name`</td><td>The new name of the subscriber group.</td></tr>
<tr><td>`t` / `topics`</td><td>The comma separated list of topic names.</td></tr>
</table>
**Examples:**
```sh
qala subscriber-groups update  <SUBSCRIBER_GROUP_NAME> -n <NEW_NAME> -d <DESCRIPTION> -t  <TOPICS_COMMA_SEPERATED_NAMES> -a <AUDIENCE>
```


### `delete`
this command deletes the subscriber group with the specified 
NAME.

**Options:**
<table>
<tr><th>Option</th><th>Description</th></tr>
</table>
**Examples:**
```sh
qala subscriber-groups delete  <SUBSCRIBER_GROUP_NAME>
```
